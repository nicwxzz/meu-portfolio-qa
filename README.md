# 📌 Projeto de Testes QA - Plataforma de E-commerce

## 📖 Sobre o projeto
Este repositório contém o planejamento, a documentação e a execução de testes realizados em uma aplicação web de e-commerce, com o objetivo de validar suas principais funcionalidades e simular um ambiente real de garantia de qualidade de software.

## 🎯 Objetivo
Garantir a qualidade do sistema por meio da validação do fluxo principal do usuário, identificando falhas e assegurando o correto funcionamento das funcionalidades críticas.

## 🧪 Escopo dos testes
Os testes contemplam os seguintes módulos:
* Login de usuário
* Listagem de produtos
* Carrinho de compras
* Processo de checkout

## 🧪 Tipos de testes aplicados
* Testes funcionais
* Testes exploratórios
* Testes de regressão

## ❌ Fora de escopo
* Testes de performance
* Testes de carga
* Testes de segurança

## 🛠 Abordagem de testes
* Testes manuais baseados em casos de teste estruturados
* Análise exploratória para identificação de cenários não previstos

## 🤖 Automação de testes
A automação de testes será implementada futuramente utilizando ferramentas como Cypress ou Playwright, com foco nos fluxos críticos da aplicação.

## 📂 Estrutura do repositório
* `/documentos` → Documentação de testes (plano, casos e bugs)
* `/evidencias` → Capturas de tela dos testes executados
* `/automacao` → Scripts de automação (em desenvolvimento)
